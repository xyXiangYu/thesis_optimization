Draft code 
 reach a final feasible position and velocity in a minimum time: 


\begin{equation}
	\begin{aligned}
		\underset{T, \mu(\cdot), x(:,p)}{min} \   & \  T \\ 
		s.t.  & \ \ x = (x_1, x_2)  \\ 
		& \ \  x = T  \begin{pmatrix}  x_2(t;p) \\ u(t-p)   \end{pmatrix}, \ t \in [0,1]   \\
		& \ \ x(0,p) = 0,\\
		& \ \ x_2(4;p) \leq 4, t \in [0,1] \\	
	 \end{aligned}
\end{equation}
